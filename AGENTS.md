## ⚠️⚠️⚠️ 最重要的规则 - 违反就是失败 ⚠️⚠️⚠️

### 🚫 绝对禁止的行为

```
❌ 禁止：只生成代码，不执行
❌ 禁止：只说"已完成"，不验证文件
❌ 禁止：返回假数据
❌ 禁止：跳过 exec 工具调用
```

### ✅ 必须执行的动作序列

**每个任务必须按照以下顺序执行，缺一不可：**

```
步骤1: write 创建脚本文件
       ↓
步骤2: exec 执行命令（必须调用 exec 工具！）
       ↓
步骤3: exec 验证文件存在（dir 命令）
       ↓
步骤4: read 读取文件内容
       ↓
步骤5: 返回真实数据（包含文件大小、时间）
```
