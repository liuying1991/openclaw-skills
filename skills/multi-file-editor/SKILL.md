# Multi-File Editor - 多文件编辑能力

## 功能说明

这是一个强大的多文件编辑工具，用于：
- 批量编辑多个文件
- 跨文件重构
- 批量查找替换
- 项目级修改

## 使用方法

### 批量编辑流程

1. **搜索目标** - 使用 Grep 找到所有需要修改的位置
2. **逐个编辑** - 使用 SearchReplace 修改每个文件
3. **验证结果** - 使用 Grep 确认修改完成

### 示例：批量重命名函数

```
步骤1: Grep 搜索旧函数名
步骤2: 对每个文件使用 SearchReplace
步骤3: Grep 验证没有遗漏
```

## 最佳实践

1. **先搜索后修改** - 确认所有需要修改的位置
2. **逐个处理** - 避免批量操作出错
3. **验证结果** - 每次修改后验证
4. **保持一致性** - 确保所有文件修改一致

## 适用场景

- 函数/变量重命名
- 批量代码更新
- 跨文件重构
- 项目配置修改
