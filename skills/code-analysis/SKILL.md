# Code Analysis - 代码分析能力

## 功能说明

这是一个强大的代码分析工具，用于：
- 分析代码结构和架构
- 理解代码逻辑和流程
- 生成代码文档
- 识别代码模式和最佳实践

## 使用方法

### 分析步骤

1. **读取代码** - 使用 read 工具读取源文件
2. **分析结构** - 识别模块、类、函数
3. **理解逻辑** - 追踪数据流和控制流
4. **生成文档** - 创建分析报告

## 分析维度

| 维度 | 说明 |
|------|------|
| 结构分析 | 模块、类、函数的组织方式 |
| 逻辑分析 | 数据流、控制流、调用关系 |
| 模式识别 | 设计模式、架构模式 |
| 质量评估 | 代码质量、可维护性 |

## 输出格式

```markdown
# 代码分析报告

## 文件结构
- 模块列表
- 类/函数列表

## 核心逻辑
- 主要功能说明
- 数据流程图

## 设计模式
- 使用的模式
- 模式说明

## 改进建议
- 优化点
- 重构建议
```

## 适用场景

- 新项目理解
- 代码审查
- 重构准备
- 文档生成
