# Context Manager - 上下文管理

## 功能说明

这是一个上下文管理工具，用于：
- 管理长对话上下文
- 提取和保存关键信息
- 记忆管理
- 信息检索

## 使用方法

### 关键信息提取

```
1. 识别关键信息（用户需求、约束条件、重要数据）
2. 提取并结构化
3. 保存到记忆系统
4. 后续对话中检索使用
```

### 上下文压缩

当对话过长时：
1. 总结已完成的工作
2. 提取关键决策
3. 记录待办事项
4. 压缩历史上下文

## 最佳实践

1. **及时记录** - 重要信息立即保存
2. **定期总结** - 长对话定期总结
3. **结构化存储** - 使用结构化格式
4. **快速检索** - 需要时能快速找到

## 适用场景

- 长对话管理
- 多会话信息共享
- 项目状态跟踪
- 知识积累
